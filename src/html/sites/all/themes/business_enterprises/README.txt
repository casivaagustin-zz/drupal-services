About business enterprise Theme
================================
Simple business enterprise is a fixed width (980px) theme.
The theme is not dependent on any core theme.
Its very light weight for fast loading with modern look.

  Simple and clean design
  Fixed width (900px)
  Drupal standards compliant
  Custom front-page with 1 block regions
  Implementation of a JS Slideshow
  Multi-level drop-down menus
  Custom front-page with 4 block regions
  Footer with 3 regions
  A total of 12 regions
  Compatible and tested on IE8, IE9+, Opera, Firefox, Chrome browsers

Browser compatibility:
=======================
The theme has been tested on following browsers. IE7+, Firefox, Chrome, Opera.

Banner images:
=======================
You can replace three banner images

Image size - width: 840px; height: 374px; 
File name - slide-image-1.jpg, slide-image-2.jpg, slide-image-3.jpg

======================
